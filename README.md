# The Greggory Foundation Ltd - Website

A modern, professional website for The Greggory Foundation Ltd, a leading consultancy that uses expert project management principles to drive business management, innovation, improvement, and successful project delivery.

## Features

- **Responsive Design**: Optimized for all devices
- **Modern UI**: Built with React, TailwindCSS, and Lucide icons
- **Professional Branding**: Navy blue, charcoal grey, with teal and gold accents
- **Complete Site Structure**: Home, About, Services, Case Studies, Blog, Contact

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/       # Reusable components
├── pages/           # Page components
├── App.jsx          # Main app component
├── main.jsx         # Entry point
└── index.css        # Global styles
```

## Technologies Used

- React 18
- Vite
- TailwindCSS
- React Router DOM
- Lucide React Icons
